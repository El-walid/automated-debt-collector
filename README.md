# Automated Debt Collector (Le Relanceur Automatique)


generate_ledger.py: A script to create a dummy Excel file (comptabilite.xlsx) with 100 fake Moroccan clients, invoice numbers, amounts, and statuses ("Payée" vs "Impayée").

pdf_generator.py: The engine that uses fpdf to draw a clean, professional PDF table for each client showing exactly what they owe in DH.

auto_relance.py: The main script that ties it together and runs the logic.
